# codedex-projects
# ☕ Minha Jornada em Java no Codédex

Este repositório foi planejado e estruturado para centralizar todos os meus desafios, códigos e algoritmos desenvolvidos na plataforma **Codédex**, com foco exclusivo na linguagem **Java**. Ele funciona como meu laboratório prático de Programação Orientada a Objetos (POO) e lógica pura, integrado ao meu ecossistema global do **PortfolioHUB**.

---

## 🎯 Objetivos do Repositório
* **Domínio da Sintaxe Java:** Praticar tipagem forte, estruturas de controle, arrays, métodos e os pilares iniciais da Orientação a Objetos.
* **Boas Práticas de Engenharia:** Aplicar convenções de nomenclatura Java (*camelCase*), organização de pacotes e tratamento básico de exceções.
* **Versionamento Seguro:** Utilizar o Git para registrar a evolução diária do meu raciocínio lógico em Java no primeiro semestre de Engenharia de Software.
